# StringFun – Java String Operations Demo

This Java program demonstrates the use of commonly used `String` methods in Java.  
It takes two strings as input from the user and performs multiple string operations to show how strings can be manipulated, compared, and analyzed.

---

## What the Program Does
- Accepts two strings from the user using the `Scanner` class
- Performs different string operations on the inputs
- Prints the results of each operation to the console

---

## String Operations Covered
- Convert a string to lowercase and uppercase
- Find the length of a string
- Replace characters in a string
- Trim leading and trailing spaces
- Compare strings using:
  - `equals()`
  - `equalsIgnoreCase()`
- Access a character using `charAt()`
- Concatenate two strings
- Extract substrings
- Find the index and last index of characters

---

## Purpose
This program is intended for beginners learning Java.  
It helps in understanding how Java’s `String` class works and how different string methods behave with user input.
